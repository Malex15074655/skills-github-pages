# Proyecto Blog de recomendaciones de peliculas de terror 

Este proyecto es un blog de peliculas, sencillo que permite a los usuarios iniciar sesión y explorar algunos titulos de peliculas.

## Estructura de Archivos

- `login.html`: Página de inicio de sesión.
- `dashboard.html`: Página principal con enlaces a las peliculas.
- `destino1.html`, `destino2.html`, `destino3.html`,'destino4',destino5': Páginas individuales para cada pelicula recomendada.
- `assets/`: Carpeta que contiene los recursos del proyecto.
  - `css/`: Contiene el archivo `style.css` para el estilo del blog.
  - `js/`: Contiene el archivo `login.js` para la funcionalidad de inicio de sesión.
  - `img/`: Carpeta que contiene las imágenes utilizadas en el blog.

## Instalación

1. Clona este repositorio o descarga los archivos.
2. Abre `login.html` en tu navegador.

## Uso

- Ingresa un nombre de usuario y contraseña para acceder al blog (admin,12345).
- Explora diferentes peliculas recomendadas haciendo clic en los enlaces.
